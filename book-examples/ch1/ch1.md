# Go First Steps
## Commands
- `go version`
  - `go version go1.20.5 darwin/arm64`
  - `go version go1.20.5 linux/amd64`
- Format all files `go fmt ./...`
- Scans for common coding mistakes `go vet ./...`
- `go build main.go`
- `go run main.go`
- `go mod init mod_name`
- `go get tidy`
